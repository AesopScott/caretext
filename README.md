# CareText

Static starter site for `caretext.net`, deployed to Cloudflare Pages from GitHub Actions.

## Deployment

Pushes to `main` deploy the repository root to the Cloudflare Pages project `caretext-net`.

Required GitHub Actions secrets:

- `CLOUDFLARE_API_TOKEN`
- `CLOUDFLARE_ACCOUNT_ID`

Production URLs:

- https://caretext.net
- https://www.caretext.net
- https://caretext-net.pages.dev
